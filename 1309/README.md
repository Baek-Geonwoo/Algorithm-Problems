### 접근 방식
- ```P[n]는 2xn 크기의 사자우리에 사자를 배치하는 경우의 수```
- 아래 4개의 ```P[n] 값으로부터 다음의 DP 점화식을 얻을 수 있다. P[n] = P[n-1]*2 + P[n-2]```
    + ```P[1] = 3```
    + ```P[2] = 7```
    + ```P[3] = 17```
    + ```P[4] = 41```
---

### 소스코드
- 메모리 : 33692KB
- 시간 : 112ms
```Python
import sys
n = int(sys.stdin.readline())
P = [0]*(n+1)
P[1] = 3
if n >= 2:
    P[2] = 7
for i in range(3,n+1):
    P[i] = (P[i-1]*2 + P[i-2])%9901
print(P[n])
```
